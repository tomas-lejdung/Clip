# Clip 1.3.8

Clip 1.3.8 makes the menu bar experience faster, more reliable, and more
consistent across recording and Live Share.

## Menu bar

- Switch between Clip's normal, recording, Live Share, and viewer screens
  inside one stable popover, avoiding delayed focus and first-click misses.
- Expand each menu row's real click target so capture, display, and recent
  recording actions respond reliably.
- Keep the normal menu fully visible after returning from Live Share.
- Replace the nested recording card with a compact, flat recording status
  layout that belongs directly to the popover.

## Live Share

- Make Native the default Web Viewer scale mode.
- Show a trusted friend's saved name in the viewer list after their identity
  has been cryptographically verified.
- Simplify the pre-share joining guidance and remove implementation-specific
  wording from the Friends section.

## Requirements

Clip requires macOS 15 or later on Apple silicon. This build uses the same
stable Apple Development identity as previous updater releases.
