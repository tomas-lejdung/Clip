# Clip 1.3.1

Clip 1.3.1 fixes native Live Share invite and Friend joins.

## Live Share

- Accept the host's signed native session descriptor after the peer-to-peer
  control channel opens instead of incorrectly reporting that the Live Share
  server returned an invalid message.
- Apply the fix to both complete-invite joins and saved-Friend joins.

The Live Share server does not need to be updated for this fix. A 1.3.1 viewer
can join a host running Clip 1.3.0 or later.

## Requirements and first launch

Clip requires macOS 15 or later on Apple silicon.

This build is Apple Development signed rather than Developer ID signed and
notarized. A direct browser download may therefore require **Open Anyway** in
**System Settings → Privacy & Security**.
