# Clip 1.3.4

Clip 1.3.4 gives native Live Share windows a cleaner CoScreen-inspired frame
and adds a true Native 100% viewing mode.

## Live Share

- Replace the receiver's visible macOS title bar and redundant source chip with
  a thick friend-colored outer frame and matching draggable header.
- Keep viewer controls translucent while idle and make them fully visible on
  hover.
- Add Fit to Window and Native 100% modes, a zoom indicator, and a command to
  reset the window to its actual logical size.
- Preserve the shared window at 100% in Native mode. Oversized content is
  cropped and smoothly follows the focused remote cursor without exposing
  blank space.
- Capture and display the cursor only for the focused shared source, including
  protection against delayed cursor messages after focus changes.
- Preserve source-logical sizing across Retina and non-Retina displays and keep
  resized viewer windows safely inside the visible screen.

This is an app-only update. The Live Share server and web viewer are unchanged.

## Requirements

Clip requires macOS 15 or later on Apple silicon. This build uses the same
stable Apple Development identity as previous updater releases.
