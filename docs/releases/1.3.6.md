# Clip 1.3.6

Clip 1.3.6 adds smoother, configurable cursor updates to Live Share.

## Live Share

- Add a **Match Cursor to Frame Rate** toggle to the active Live Share popover
  and the default Live Share settings.
- Keep the existing 20 Hz cursor cadence when the option is disabled.
- Follow 30 or 60 FPS stream cadence when enabled, while retaining a 20 Hz
  minimum for 15 FPS streams.
- Apply changes immediately during an active share and remember the setting for
  future sessions.
- Keep cursor messages ephemeral so stale positions are discarded under
  transport congestion instead of being queued.

This is an app-only update. The Live Share server and web viewer are unchanged.

## Requirements

Clip requires macOS 15 or later on Apple silicon. This build uses the same
stable Apple Development identity as previous updater releases.
