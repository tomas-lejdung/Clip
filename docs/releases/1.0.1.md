# Clip 1.0.1

This maintenance release verifies Clip's signed in-app update path from version
1.0.0. It contains the same recording, preview, export, History, and Settings
features as the initial release.

## Updates

Clip 1.0.0 can discover this release automatically or through **Check for
Updates…** in the menu-bar popover. The updater verifies the download with the
public EdDSA key embedded in Clip before installing it.

## Requirements and first launch

Clip requires macOS 15 or later on Apple silicon.

This build is Apple Development signed but is not Developer ID signed or
notarized. A direct browser download may therefore require **Open Anyway** in
**System Settings → Privacy & Security**. This limitation will remain until a
Developer ID–signed and notarized release is available.
