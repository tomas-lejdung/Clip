# Clip 1.3.5

Clip 1.3.5 tightens and simplifies the custom native Live Share window header.

## Live Share

- Reduce the shared-window header height so it uses less vertical space.
- Remove the shared translucent background behind the viewer controls.
- Render Zoom and Close as separate black controls with independent hover
  feedback, so hovering one no longer highlights both.
- Keep the zoom icon and current percentage together as one button.

This is an app-only update. The Live Share server and web viewer are unchanged.

## Requirements

Clip requires macOS 15 or later on Apple silicon. This build uses the same
stable Apple Development identity as previous updater releases.
