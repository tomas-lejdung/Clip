# Clip 1.3.7

Clip 1.3.7 adds live, codec-specific stream tuning and more predictable color
for Live Share.

## Live Share

- Add advanced controls for VP8, VP9, AV1, and H.264, available from both
  Settings and an active Live Share session.
- Apply advanced changes to the current stream immediately, preview the result,
  reset one codec to Clip's defaults, and persist each codec independently.
- Add Compatible Rec.709, Full-range Rec.709, and experimental Native Display
  color modes.
- Preserve Rec.709 color metadata through AV1 encoding, decoding, and Clip's
  native Metal viewer for more accurate color and flatter solid-color regions.
- Keep H.264's VideoToolbox-specific quality, QP, and keyframe controls while
  exposing the applicable WebRTC sender controls consistently for every codec.

## Dependency update

- Use Clip's reviewed WebRTC M150 build with an auditable source patch,
  immutable binary checksum, and complete bundled third-party notices.

The Live Share server protocol and web viewer are unchanged.

## Requirements

Clip requires macOS 15 or later on Apple silicon. This build uses the same
stable Apple Development identity as previous updater releases.
